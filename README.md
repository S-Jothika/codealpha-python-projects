# 🚀 CodeAlpha Internship Projects

## 👩‍💻 About This Repository
This repository contains the projects completed as part of the **CodeAlpha Internship Program**.  
It includes 4 Python-based beginner projects that focus on improving programming fundamentals, logic building, and real-world problem-solving skills.

---

## 📂 Projects Overview

### 1️⃣ Task 1: Hangman Game
A text-based word guessing game where the player tries to guess the hidden word one letter at a time with limited attempts.

---

### 2️⃣ Task 2: Stock Portfolio Tracker
A simple program that calculates total investment based on user input and predefined stock prices stored in a dictionary.

---

### 3️⃣ Task 3: AI Chatbot using Gemini API
An AI-powered chatbot that uses Google Gemini API to generate intelligent and dynamic responses in real time.

---

### 4️⃣ Task 4: Basic Rule-Based Chatbot
A simple chatbot that responds to predefined user inputs like "hello", "how are you", and "bye" using conditional statements.
